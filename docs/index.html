Documentation for the BFB reactor model

Gavin Wiggins and Cornelius Agu

Updated on May 7, 2021


This documentation is for the one-dimensional bubbling fluidized bed (BFB) reactor model developed in Python. The model predicts dynamic and steady-state conditions of a BFB reactor. See the sections below for usage information along with discussions of the equations that represent the model.

The Python code for the model is open source and available on GitHub.

Fluidization

This section discusses the minimum fluidization velocity, bed expansion, and other equations related to the hydrodynamics of a bubbling fluidized bed reactor.

Minimum fluidization velocity

The minimum fluidization velocity is calculated as follows $$U_{mf} = \frac{Re\, \mu}{\rho_g\, d_p}$$ $$Re = -33.67 + (33.67^2 + 0.0408 Ar)^{0.5}$$ $$Ar = \frac{d_p^3\, \rho_g (\rho_p - \rho_g) g}{\mu^2}$$

Nomenclature

  • \(Ar\) [-] = Archimedes number
  • \(Re\) [-] = Reynolds number
  • \(U_{mf}\) [m/s] = minimum fluidization velocity
  • \(d_p\) [m] = particle diameter
  • \(g\) [m/s²] = acceleration due to gravity
  • \(\rho_g\) [kg/m³] = gas density
  • \(\rho_p\) [kg/m³] = particle density
  • \(\mu\) [Pa⋅s] = dynamic gas viscosity
